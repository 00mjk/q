インストール

現在の安定版は 1.5.0 です。あなたの望むパッケージを選んでください:

プラットフォーム インストール方法 コメント
​Mac brew update を実行後に brew install q を実行してください。
​RPM パッケージ パッケージをダウンロードした後、 rpm -ivh <package-filename> を実行。もし古いバージョンの q がインストールされている場合は、 rpm -U <package-filename> を実行してください。 このリリースバージョンでは man が使用できます。 man q を叩くだけです。
​Debian パッケージ パッケージをダウンロードした後、 sudo dpkg -i <package-filename> このリリースバージョンでは man が使用できます。 man q を叩くだけです。
​Windows インストーラー インストーラーを起動し、「次へ」を連打してください。 q.exe が PATH に追加されるのでどこからでも実行可能です。 Windows は PATH に追加しただけでは、コマンドが反映されません。cmd を立ち上げ直す必要があります。
​単体ファイル パスが通っている所に配置し、 chmod +x q コマンドで実行権限を付加してください。 Python 2.5 以上が必要です。 もし python 2.4 を使用しているなら、pip install sqlite を実行するだけで構いません。
​Arch Linux installing packages を見るか、お気に入りの AUR helper を使用してください。 リンク先は PKGBUILD についての説明です。
