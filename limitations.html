Limitations

  • CTE (WITH alias (SELECT ...) SELECT ... FROM alias) is not supported
  • FROM <subquery> is not supported
  • Spaces in file names are not supported yet. Use stdin for piping the data into q, or rename the file
  • Some rare cases of subqueries are not supported yet. Please open an issue if you find such a case.
